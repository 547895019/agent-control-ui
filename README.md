# OpenClaw Agent Control UI

A React SPA for managing and visualizing OpenClaw Agent Gateway.

## Features
- Direct WebSocket/HTTP connection to Gateway
- Agent Management (Create, Edit, Delete)
- Built-in Editor (Monaco Editor)
- Live Gateway Monitor

## Stack
- React 18
- TypeScript
- Vite
- Zustand
- TailwindCSS
- Monaco Editor

## Run Locally
```bash
npm install
npm run dev
```

Point to `ws://127.0.0.1:18789` and `http://127.0.0.1:18789`
