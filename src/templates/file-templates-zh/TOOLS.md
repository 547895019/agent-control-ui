# TOOLS.md - 本地配置笔记

技能定义了工具_怎么用_，这个文件记录的是_你自己的_具体配置——你的环境里独有的东西。

## 记什么

比如：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 偏好语音
- 音箱/房间名称
- 设备昵称
- 任何环境特有的信息

## 示例

```markdown
### 摄像头

- 客厅 → 主区域，180° 广角
- 门口 → 入口，运动触发

### SSH

- 家庭服务器 → 192.168.1.100，用户名：admin

### 语音合成

- 偏好语音："晓晓"（温暖，自然）
- 默认音箱：客厅HomePod
```

## 为什么单独放一个文件？

技能是共享的，你的配置是你自己的。分开存放意味着你可以更新技能而不丢失笔记，也可以分享技能而不泄露你的基础设施信息。
