# TOOLS.md - 本地备注

技能定义了工具_如何_工作。这个文件是用于_你的_具体情况 —— 那些对你的设置独一无二的东西。

## 这里放什么

比如：

- 摄像头名称和位置
- SSH 主机和别名
- TTS 的首选声音
- 扬声器/房间名称
- 设备昵称
- 任何特定于环境的内容

## 示例

```markdown
### 摄像头

- living-room → 主区域，180° 广角
- front-door → 入口，运动触发

### SSH

- home-server → 192.168.1.100，用户: admin

### TTS

- 首选声音: "Nova"（温暖，略带英音）
- 默认扬声器: 厨房 HomePod
```

## 为什么分开？

技能是共享的。你的设置是你的。把它们分开意味着你可以更新技能而不会丢失你的备注，并且可以分享技能而不会泄露你的基础设施。
